### PuleF32m33

matrices are 'column major', the indices look like

   |  0  4  8 12 |
   |  1  5  9 13 |
   |  2  6 10 14 |
   |  3  7 11 15 |

which compose vectors
  <0 .. 3>, <4 .. 7>, <8 .. 11>, <12 .. 15>

for example, the 'translation' part of a view matrix would be stored at
   <12, 13, 14, 15>

```c
struct {
  elem : float [9];
};
```
### PuleF32m44
```c
struct {
  elem : float [16];
};
```
